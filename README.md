This is a fully responsive template. Based on **[Bootstrap 3](https://github.com/twbs/bootstrap)** framework. Highly customizable and easy to use. Fits many screen resolutions from small mobile devices to large desktops. Check out the live preview now and see for yourself.

**Download & Preview on [EONConsulting](https://github.com/EonConsulting/UnisaTemplate)**

!["Unisa Presentation "](https://github.com/EonConsulting/UnisaTemplate/blob/master/dist/img/UnisaShot.png)

This template has been carefully coded with clear comments in all of its JS, LESS and HTML files. LESS has been used to increase code customizability.

Installation
------------
There are multiple ways to install this template.

####Download:

Download from Github and download the latest release.

####Using The Command Line:

**Github**

- Fork the repository ([here is the guide](https://help.github.com/articles/fork-a-repo/)).
- Clone to your machine
```
git clone https://github.com/YOUR_USERNAME/UnisaTemplate.git
```

**Bower**

```
bower install admin-lte
```

**npm**

```
npm install --save admin-lte
```

**Composer**

```
composer require "almasaeed2010/adminlte=~2.0"
```
